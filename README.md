## js-math

**A lightweight Javascript time format library.**

### Installing

```bash
npm install @cdjs/js-time
// or
yarn add @cdjs/js-time
```

### Usage

```javascript
// es6 module
import jsTime from '@cdjs/js-time'
// commonjs
const { default: jsTime } = require('@cdjs/js-time')
```
